# c2cscrape.py

Uses beautifulsoup4 to scrape torrent links from knaben.org for C2C AM. Sends these torrents to qbittorrent.

In order to use this script, you must have qbittorrent running on your local machine or a server and have qbittorrent credentials set in the .env file.

By default adds last 5 torrents to qbittorrent.
